# Praise Bucuzong Bah - Portfolio Website

A modern, futuristic portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- 🎨 **Dark theme** with warm browns, golds, and bold red accents
- ✨ **Glassmorphism effects** and smooth animations (Framer Motion)
- 📱 **Fully responsive** design (mobile-first)
- 🚀 **Fast performance** with Vite build tool
- 📝 **Interactive sections**: Hero, About, Skills, Services, Portfolio, Experience, Testimonials, Contact

## Tech Stack

- **React 18** + TypeScript
- **Vite** for fast development
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **React Hook Form** for form validation
- **Lucide React** for icons

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Add your profile image
Copy your profile photo to:
```
public/profile.jpg
```

### 3. Run development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```

## Project Structure

```
src/
├── components/     # React components (Navigation, Hero, About, etc.)
├── data/          # Data files (projects, skills, services, experience)
├── App.tsx        # Main app component
├── main.tsx       # Entry point
└── index.css      # Global styles with Tailwind

public/
├── profile.jpg    # Your profile photo
├── favicon.svg    # Site favicon
└── cv.pdf         # Your CV (optional)
```

## Customization

- **Colors**: Edit `tailwind.config.js` to change the color scheme
- **Content**: Edit files in `src/data/` to update projects, skills, etc.
- **Sections**: Components are in `src/components/`
